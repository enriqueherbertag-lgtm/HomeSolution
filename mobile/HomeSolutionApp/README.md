#HomeSolution Mobile App

> Aplicación React Native para la plataforma HomeSolution - Conectando técnicos con clientes

## Estado Actual
**Versión:** 1.0.0-alpha  
**Plataforma:** React Native + Expo  
**Estado:** En desarrollo activo

## Características Implementadas
- Pantalla de Login/Registro
- Selección Cliente/Técnico
- Diseño UI/UX profesional
- Estructura de navegación básica

## Próximos Hitos
1. **Pantalla de Mapa** - Geolocalización de técnicos
2. **Conexión API** - Autenticación JWT + endpoints
3. **Chat en tiempo real** - WebSockets integrados
4. **Sistema de pagos** - Stripe/MercadoPago
5. **Notificaciones push** - Firebase Cloud Messaging
