# Guía de Contribución

## IMPORTANTE
Este es un proyecto **propietario**. Solo se aceptan contribuciones de:

1. **Colaboradores autorizados previamente**
2. **Pull Requests con acuerdo de licencia firmado**
3. **Correcciones de bugs menores previamente discutidas**

## Proceso
1. Abrir un **issue** describiendo el cambio
2. Esperar aprobación del mantenedor
3. Firmar acuerdo de contribución (si aplica)
4. Hacer Pull Request a la rama `develop`
